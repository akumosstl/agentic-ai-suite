# Templates Guide

This guide explains how to use templates for quick starting entities in Agentic.

## Accessing Templates

Go to **Management > Templates** in your browser.

## Overview

Templates provide starting points for different entity types:

| Type | Description |
|------|-------------|
| Agents | Prompt templates |
| Scripts | Script templates |
| Instructions | Instruction templates |

## Using Templates

When creating an entity:
1. Select a template from the dropdown
2. Template content populates the form
3. Modify as needed
4. Save the entity

## Creating a Template

1. Click **Add**
2. Select template type
3. Fill in name, description, and content
4. Click **Create Template**

## Notes

- Templates are type-specific
- Help quickly start common patterns
