# Pipeline Executions Guide

This guide explains how to view and manage all pipeline runs in Agentic.

## Accessing Pipeline Executions

Go to **Stack > Pipelines** in your browser.

## Overview

The Pipelines screen shows all pipeline runs:

| Panel | Description |
|-------|-------------|
| Left Panel | List of all runs with search |
| Right Panel | Pipeline flow visualization |

## Viewing Runs

### Run List

Each run shows:
- Pipeline name
- Project name
- Date
- Status indicator

### Run Details

Click on any run to view its details.

### View Running Pipeline

For running pipelines, click the eye icon to view live execution.

## Status Indicators

| Status | Meaning |
|--------|---------|
| Completed | Finished successfully |
| Running | Currently executing |
| Failed | Execution failed |
| Pending | Waiting to execute |

## Searching

Type in the search box to filter by project name.

## Cleanup History

Click **Cleanup history** to delete old completed and failed runs.

## Notes

- Shows all runs across all projects
- Running pipelines can be viewed in real-time
- Cleanup only removes non-running pipelines
