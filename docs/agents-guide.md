# Agent Management Guide

This guide explains how to create, manage, and organize AI agents in Agentic.

## Accessing Agents

Go to **Stack > Agents** in your browser.

## Overview

The Agents screen has two panels:

| Panel | Description |
|-------|-------------|
| Left Panel | Agent list with search and pagination |
| Right Panel | Agent details form |

## Creating an Agent

### Step 1: Click Add

Click the "Add" button to start a new agent.

### Step 2: Fill in Details

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Agent identifier |
| Namespace | No | Category grouping |
| Prompt | Yes | Instructions for the agent |

### Step 3: Edit the Prompt

Click "Open in Editor" to use the built-in editor with formatting.

### Step 4: Save

Click **Create Agent** to save.

## Searching Agents

- Search by name
- Filter by namespace
- Clear filters with X button
- Pagination for navigation

## Editing an Agent

1. Select an agent from the list
2. Modify fields
3. Click **Update Agent**

## Deleting an Agent

1. Select an agent
2. Click the delete icon
3. Confirm deletion

## Copying Prompt

Click "Copy to clipboard" to copy the agent's prompt.

## Notes

- Names must be unique
- Namespaces help organize agents
- Templates available for quick start
