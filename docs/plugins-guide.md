# Plugins and Tools Guide

This guide explains how to use Plugins (Tools) in Agentic.

## Accessing Tools

Click **Tools** in the menu bar.

## Overview

The Tools modal provides extensions that extend system capabilities.

## Available Features

| Feature | Description |
|---------|-------------|
| Browse Tools | View and install available plugins |
| My Plugins | View your installed plugins |
| Search | Find specific plugins |

## Notes

- Plugins extend system capabilities
- Can be searched and installed from the registry
