# Namespaces Guide

This guide explains how to organize entities by namespace in Agentic.

## Accessing Namespaces

Go to **Management > Namespace** in your browser.

## Overview

Namespaces organize entities by category.

## Viewing Namespaces

### Namespace List

Click on any namespace to view its dashboard showing:
- Items in this namespace
- Pipelines using this namespace
- Projects using this namespace

## Using Namespaces

When creating an entity:
1. Assign a namespace (optional)
2. Entity is grouped by category

## Clearing a Namespace

Click **Clear** to remove items not in use by pipelines or projects.

## Notes

- Namespaces help organize entities
- Protected items (in use) cannot be cleared
