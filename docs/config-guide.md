# Configuration Guide

This guide explains how to configure application settings in Agentic.

## Accessing Configuration

Go to **Config** menu option.

## Overview

The Configuration screen manages targets and resource paths.

## Managing Targets

Targets define where resources are stored.

### Creating a Target

1. Click **+** to add new target
2. Fill in paths for each resource type
3. Click **Create**

### Editing a Target

1. Click on a target row
2. Modify path fields
3. Click **Update**

### Deleting a Target

1. Click delete icon
2. Confirm deletion

## Notes

- Targets define file paths for resources
- Default target is created automatically
