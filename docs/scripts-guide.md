# Scripts Management Guide

This guide explains how to create and manage executable scripts in Agentic.

## Accessing Scripts

Go to **Stack > Scripts** in your browser.

## Overview

The Scripts screen allows you to create executable scripts:

| Panel | Description |
|-------|-------------|
| Left Panel | Script list with search |
| Right Panel | Script details form |

## Creating a Script

### Step 1: Click Add

Click the "Add" button to start a new script.

### Step 2: Fill Details

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Script identifier |
| Namespace | No | Category grouping |
| Script | Yes | The script code |

### Step 3: Edit Script

Click "Open in Editor" for better editing experience.

### Step 4: Save

Click **Create Script** to save.

## Using Scripts in Pipelines

Scripts can be added as steps in pipelines:
- Script code runs during execution
- Output is captured
- Exit code determines success/failure

## Editing a Script

1. Select script from list
2. Modify fields
3. Click **Update Script**

## Deleting a Script

1. Select script
2. Click delete icon
3. Confirm deletion

## Copy Script to Clipboard

Click "Copy" to copy the script code.

## Notes

- Scripts contain executable code
- Templates available for quick start
- Namespace organizes scripts
