# Instructions Management Guide

This guide explains how to create and manage instruction sets in Agentic.

## Accessing Instructions

Go to **Stack > Instructions** in your browser.

## Overview

The Instructions screen allows you to create reusable instruction sets:

| Panel | Description |
|-------|-------------|
| Left Panel | Instruction list with search |
| Right Panel | Instruction details form |

### Unique Feature: Instruction Files

Instructions can include **files** - supporting documents or rules.

## Creating an Instruction Set

### Step 1: Click Add

Click the "Add" button to start a new instruction set.

### Step 2: Fill Details

| Field | Required | Description |
|-------|----------|-------------|
| Name | Yes | Instruction identifier |
| Namespace | No | Category grouping |
| Instructions | Yes | The instruction content |

### Step 3: Add Files (Optional)

1. Click **Add File** button
2. Enter file details
3. Click Add

### Step 4: Save

Click **Create Instruction** to save.

## Using Instructions in Pipelines

Instructions can be added as steps in pipelines:
- Instructions are loaded as rules
- Files provide supporting data
- Define behavior for subsequent steps

## Editing Instructions

1. Select instruction from list
2. Modify fields
3. Click **Update Instruction**

## Deleting Instructions

1. Select instruction
2. Click delete icon
3. Confirm deletion

## Notes

- Instructions define rules/guidelines
- Files provide supporting data
- Namespace organizes instruction sets
