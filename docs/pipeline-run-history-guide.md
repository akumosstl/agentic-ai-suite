# Pipeline Run History Guide

This guide explains how to view past pipeline executions in Agentic.

## Accessing Pipeline Run History

Navigate to `/pipeline-run-history` in your browser.

## Overview

The Run History screen shows past executions:
- Run list with status indicators
- Pipeline flow visualization
- Step details and output

## Viewing Runs

### Run List

Each run shows:
- Status icon
- Run number (newest first)
- Date
- Status (completed/failed/running/pending)

### Selecting a Run

Click any run to view its details.

### Pipeline Flow

View the pipeline flow with step cards showing:
- Step number
- Step type icon
- Step name
- Status colors

## Step Details

Click any step to view its details and output.

### Output Button

Click **Output** to open a modal with:
- Console output
- Error messages
- Stack traces

## Notes

- Runs are sorted newest-first
- Each run shows complete flow visualization
- Step output available via Output button
