# Instruções
1. **Diagnóstico**: Analise o erro ou comportamento inesperado que descrevi: "$1".
2. **Consulta Interna**: Pesquise em `@.agentic/brain/wiki/troubleshooting.md` e `@.agentic/brain/wiki/concepts/` se já enfrentamos algo parecido ou se há padrões recomendados.
3. **Proposta de Correção**:
   - Se encontrou algo na Wiki: "Baseado em um erro anterior (registrado em [data]), a solução pode ser..."
   - Se não encontrou: Proponha uma solução baseada em boas práticas de Spring Boot.
4. **Aplicação**: Pergunte se desejo que você aplique a correção diretamente nos arquivos de código.
5. **Ciclo de Aprendizado**: Após a correção, se funcionar, lembre-me de atualizar o `@.agentic/brain/wiki/troubleshooting.md` caso seja um erro novo.
